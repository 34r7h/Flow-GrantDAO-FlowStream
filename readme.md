## Project: FlowStream
**Tagline**: Continuous Grant Streaming using Flow EVM.

### Description
FlowStream allows DAOs and grant programs to stream funds to grantees in real-time, leveraging the low cost and high throughput of Flow EVM. This ensures accountability (streams can be stopped) and capital efficiency.

### Tech Stack
- **Blockchain**: Flow EVM
- **Smart Contracts**: Solidity (Foundry)
- **Frontend**: Next.js, RainbowKit, Wagmi, Viem
- **Indexer**: The Graph (optional/TBD)
